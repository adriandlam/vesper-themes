# Vesper Themes

Community ports of the [Vesper](https://github.com/raunofreiberg/vesper) dark theme by [Rauno Freiberg](https://github.com/raunofreiberg) for terminal tools.

Peppermint and orange flavored.

## Ports

| Tool | Repository | Description |
| ---- | ---------- | ----------- |
| [bat](https://github.com/sharkdp/bat) | [vesper-bat](https://github.com/adriandlam/vesper-bat) | TextMate theme (`.tmTheme`) for bat & Sublime Text |
| [fzf](https://github.com/junegunn/fzf) | [vesper-fzf](https://github.com/adriandlam/vesper-fzf) | Sourceable `--color` config for fzf |
| [delta](https://github.com/dandavison/delta) | [vesper-delta](https://github.com/adriandlam/vesper-delta) | Includable gitconfig for delta diff viewer |
| [eza](https://github.com/eza-community/eza) | [vesper-eza](https://github.com/adriandlam/vesper-eza) | Full `theme.yml` with file type, permission, and git colors |
| [atuin](https://github.com/atuinsh/atuin) | [vesper-atuin](https://github.com/adriandlam/vesper-atuin) | Theme TOML for atuin shell history TUI |
